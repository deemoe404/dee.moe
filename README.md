# dee.moe

Personal blog by deemoe. Built with Hexo and the claudia theme by Haojen. Live at https://dee.moe

## About

This repository hosts the source of my blog. It’s a mix of projects, notes, essays, and assorted writings. Expect posts on:

- Creation: hands‑on projects and showcases
- Technology: build logs, fixes, scripts, and how‑tos
- Essays & Writings: thoughts, observations, and short pieces

Navigation, style, and widgets come from the claudia theme; comments are powered by utterances. Content is organized by folders and dates with clean permalinks.

## Tech at a glance

- Hexo 7 with Pug/Marked/SCSS renderers
- Theme: `themes/claudia`
- Search and RSS enabled
- Post asset folders enabled (images/videos live alongside each post)
- Light/dark appearance via theme
